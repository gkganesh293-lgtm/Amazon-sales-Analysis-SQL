# Amazon Sales Analysis using SQL

## Project Overview
Analyzed 14,290 Amazon sales records using MySQL and SQL queries to derive business insights.

## Key Findings
- Total Orders: 14,290
- Total Revenue: 8,928,994.98
- Highest Revenue State: Maharashtra
- Highest Revenue Category: Set
- Majority orders were successfully shipped
